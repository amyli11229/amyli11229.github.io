Soil Science Viewer — Gaussian Splat Visualization

EML Research & Visualization Project Figma, Gaussian Splatting, GitHub

A scientific visualization project focused on creating an accessible viewer for Gaussian Splat-based soil scans. The project aimed to simplify the interaction experience for complex spatial data and support educational and research-based use cases.

As the sole UI/UX designer on the project, I designed the interface system and optimized the overall user flow for navigating 3D soil visualization models. I worked closely with developers to improve usability, interaction clarity, and accessibility for users unfamiliar with technical visualization tools.

The project emphasized clean interface design, intuitive navigation, and translating complex data into a more approachable interactive experience.

Key contributions

  • Designed interface layouts and interaction systems
  • Optimized user flow and navigation experience
  • Collaborated directly with developers during implementation
  • Improved accessibility and usability of scientific visualization tools
  • Created UI prototypes and design iterations in Figma
